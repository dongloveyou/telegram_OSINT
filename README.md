Kho lưu trữ telegram_OSINT chứa công cụ hỗ trợ thu thập và phân tích thông tin tình báo nguồn mở (OSINT)
Thu thập thông tin từ tài khoản, nhóm (Groups) hoặc kênh (Channels) công khai
Phân tích dữ liệu, lịch sử tin nhắn hoặc các siêu dữ liệu (metadata) liên quan

1. Chuẩn bị môi trường
2. Đảm bảo hệ thống của bạn đã cài đặt Python 3.8+ (Khuyến nghị sử dụng các bản phân phối Linux như Kali Linux hoặc cài đặt Python trên Windows)
3. Cài đặt thư viện phụ thuộc
